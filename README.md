# AXION 🛍️

Modern full-stack e-commerce frontend built with Next.js, TypeScript, and Tailwind CSS.

---

## 🚀 Overview

AXION is a modern e-commerce web application focused on performance, clean UI, and scalable architecture.  
It is designed as a production-ready frontend for a real-world online store.

---

## ✨ Features

- Modern responsive UI/UX
- Product listing system
- Product details pages
- Shopping cart functionality
- Wishlist system
- Static pages:
  - Privacy Policy
  - Terms of Service
  - Shipping Policy
  - Returns Policy

---

## 🛠️ Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Zustand (State Management)
